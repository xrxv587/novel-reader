# 小说阅读器 UI 设计方案

> 设计风格：参考微信读书，简洁、素雅、沉浸式阅读体验
> 平台：iOS / Android（uni-app 跨端）

---

## 一、设计原则

### 1.1 核心风格

| 原则 | 说明 |
|------|------|
| 简洁素雅 | 大面积留白，减少视觉干扰，让内容成为主角 |
| 沉浸式阅读 | 阅读页尽量隐藏 UI 元素，全屏沉浸 |
| 温暖质感 | 米白/浅灰为主色调，低饱和度，护眼舒适 |
| 层级清晰 | 通过阴影、间距、字号建立明确的视觉层级 |

### 1.2 参考基准

- 整体配色和布局参考**微信读书**
- 书架采用**三列网格**布局，卡片式展示
- 阅读页采用**滚动模式**，类似微信读书的上下滚动阅读
- 底部 tab 共 2 个：书架 / 功能

## 二、色彩系统

### 2.1 主色调

| 色值 | 用途 | 色板 |
|------|------|------|
| `#2D2D2D` | 主文字色（标题、正文） | 深灰黑 |
| `#666666` | 次级文字（说明、辅助信息） | 中灰 |
| `#999999` | 辅助文字（时间、标签） | 浅灰 |
| `#B2B2B2` | 占位文字、禁用态 | 更浅灰 |
| `#F5F5F5` | 页面背景色 | 浅灰白 |
| `#FFFFFF` | 卡片背景、导航栏背景 | 纯白 |
| `#2E7D5B` | 主题色（微信绿） | 墨绿 |
| `#E8F5EE` | 主题色浅色（选中态背景） | 浅绿 |

### 2.2 阅读主题色

提供 5 种阅读背景，用户可切换：

| 名称 | 背景色 | 文字色 | 适用场景 |
|------|--------|--------|----------|
| 米白 | `#F6F2E9` | `#3E3B32` | 默认，最舒适 |
| 纯白 | `#FFFFFF` | `#2D2D2D` | 明亮环境 |
| 护眼绿 | `#CCE8CF` | `#2D5A32` | 长时间阅读 |
| 夜间黑 | `#1A1A1A` | `#999999` | 夜间/暗环境 |
| 羊皮纸 | `#F0E6D3` | `#5C4B32` | 复古质感 |

### 2.3 功能色

| 色值 | 用途 |
|------|------|
| `#E74C3C` | 错误、删除、警告 |
| `#F39C12` | 提醒、进度 |
| `#27AE60` | 成功、完成 |

## 三、字体规范

### 3.1 字体栈

```css
font-family: -apple-system, BlinkMacSystemFont, "PingFang SC", "Helvetica Neue", "Microsoft YaHei", sans-serif;
```

- iOS：优先用苹方（PingFang SC）
- Android：优先用系统默认，兜底思源黑体
- 阅读字体可切换：系统默认 / 宋体 / 楷体 / 黑体

### 3.2 字号层级

| 层级 | 字号 | 行高 | 用途 |
|------|------|------|------|
| 大标题 | 20px | 28px | 页面标题（书架、设置等） |
| 一级标题 | 17px | 24px | 书名、列表主标题 |
| 二级标题 | 15px | 22px | 章节标题、功能项标题 |
| 正文 | 14px | 20px | 普通文本、说明文字 |
| 辅助文字 | 12px | 17px | 时间、标签、小字说明 |
| 阅读正文 | 18px（默认） | 1.75倍 | 阅读页正文（用户可调 12~28px） |

### 3.3 字重

- `500`（Medium）：标题、选中态
- `400`（Regular）：正文、普通文本

## 四、间距与圆角

### 4.1 间距系统

采用 4px 基础单位：

| 间距 | 数值 | 用途 |
|------|------|------|
| xs | 4px | 元素内部微调 |
| sm | 8px | 小间距（图标与文字、标签间距） |
| md | 12px | 卡片内边距、列表项内间距 |
| lg | 16px | 页面左右边距、模块间距 |
| xl | 24px | 大模块间距 |
| xxl | 32px | 页面顶部/底部大留白 |

### 4.2 圆角

| 级别 | 数值 | 用途 |
|------|------|------|
| 小圆角 | 4px | 按钮、标签、小卡片 |
| 中圆角 | 8px | 书籍卡片、弹窗、输入框 |
| 大圆角 | 12px | 底部面板、大卡片 |
| 全圆角 | 50% | 头像、圆形按钮 |

## 五、页面设计

### 5.1 书架页（首页）

**布局：顶部导航 + 书籍网格 + 底部 tab**

```
┌─────────────────────────────────────┐
│  书架                      管理  +  │  ← 导航栏
├─────────────────────────────────────┤
│                                     │
│  ┌──────┐  ┌──────┐  ┌──────┐     │
│  │ 封面 │  │ 封面 │  │ 封面 │     │  ← 三列网格
│  │      │  │      │  │      │     │
│  └──────┘  └──────┘  └──────┘     │
│   书名        书名        书名     │
│   进度%      进度%      进度%      │
│                                     │
│  ┌──────┐  ┌──────┐  ┌──────┐     │
│  │ 封面 │  │ 封面 │  │  +   │     │  ← 最后一个是添加按钮
│  │      │  │      │  │ 导入 │     │
│  └──────┘  └──────┘  └──────┘     │
│   书名        书名     导入书籍    │
│                                     │
└─────────────────────────────────────┘
│  📚 书架          ⚙️ 功能         │  ← 底部 tab
└─────────────────────────────────────┘
```

**设计细节：**

- **导航栏**：左侧"书架"大标题（20px，加粗），右侧"管理"文字按钮 + "+"图标按钮
- **书籍卡片**（三列等宽）：
  - 封面图：宽高比约 3:4，圆角 8px，带轻微阴影（`0 2px 8px rgba(0,0,0,0.08)`）
  - 书名：封面下方，14px，最多显示 1 行，超出省略
  - 进度：书名下方，12px，灰色，显示阅读百分比
  - 卡片间距：水平 12px，垂直 20px
  - 页面左右边距：16px
- **添加卡片**：最后位置放一个"导入书籍"占位卡片，虚线边框，中间 + 图标
- **长按交互**：长按书籍进入管理模式，出现勾选框，可批量删除
- **空状态**：没有书时，中间显示插画 + "书架空空如也，去添加几本书吧" + 导入按钮

### 5.3 阅读页（核心页面）

**模式：滚动阅读（类似微信读书）**

#### 默认状态（沉浸式，全屏）

```
┌─────────────────────────────────────┐
│                                     │
│                                     │
│   第一章 风起                       │  ← 章节标题（可选显示）
│                                     │
│   正文内容正文内容正文内容正文       │
│   内容正文内容正文内容正文内容       │  ← 阅读正文（滚动）
│   正文内容正文内容正文内容正文       │
│   内容正文内容正文内容正文内容       │
│   正文内容正文内容正文内容正文       │
│                                     │
│   ...（可上下滚动）...              │
│                                     │
│                                     │
│  ◁ 上一章          下一章 ▷        │  ← 底部章节导航（可选）
│  第3章 / 共128章    45%             │
└─────────────────────────────────────┘
```

**设计细节：**

- **全屏沉浸**：默认隐藏状态栏、导航栏
- **正文区域**：
  - 左右边距：20px（可调）
  - 上下边距：40px
  - 字号：18px（默认，可调 12~28px）
  - 行高：1.75 倍（可调 1.2~2.5）
  - 首行缩进：2em
  - 段间距：1em
  - 背景色：米白 `#F6F2E9`（默认）
- **章节标题**：17px，加粗，段前间距 24px，段后间距 16px
- **底部导航条**：浅半透明背景，左右显示上一章/下一章按钮，中间显示章节进度
- **点击中间区域** → 呼出顶部导航和底部设置面板

#### 呼出菜单状态

```
┌─────────────────────────────────────┐
│  <  《书名》              ⋮        │  ← 顶部导航（下滑/点击出现）
├─────────────────────────────────────┤
│                                     │
│   （阅读内容，变暗）                │
│                                     │
├─────────────────────────────────────┤
│  目录   书签   夜间   设置         │  ← 底部工具栏
├─────────────────────────────────────┤
│  ◁ 上一章          下一章 ▷        │
│  第3章 / 共128章    45%             │
└─────────────────────────────────────┘
```

**顶部导航：**
- 左侧返回箭头
- 中间书名（15px，最多一行）
- 右侧更多按钮（三个点）

**底部工具栏（4 个图标按钮）：**
- 目录（列表图标）→ 弹出章节目录面板
- 书签（书签图标）→ 添加/移除书签
- 夜间（月亮/太阳图标）→ 切换夜间模式
- 设置（Aa 图标）→ 弹出排版设置面板

## 七、组件规范

### 7.1 按钮

| 类型 | 样式 | 使用场景 |
|------|------|----------|
| 主按钮 | 绿色背景 `#2E7D5B`，白色文字，圆角 8px，高度 44px | 主要操作（确认、导入） |
| 次按钮 | 白色背景，灰色边框，深色文字，圆角 8px | 次要操作（取消、返回） |
| 文字按钮 | 无背景，绿色/灰色文字 | 导航栏操作、列表项操作 |
| 图标按钮 | 仅图标，点击有反馈 | 工具栏、导航栏 |

### 7.2 输入框

- 高度：40px
- 背景：`#F5F5F5`
- 圆角：8px
- 文字：14px，`#2D2D2D`
- 占位符：14px，`#B2B2B2`
- 左右内边距：12px

### 7.3 弹窗 / ActionSheet

- 底部弹出，圆角 12px（顶部两个角）
- 背景：白色
- 标题：16px，加粗，居中
- 选项：高度 48px，15px 文字，居中或左对齐
- 取消按钮：和选项之间有 8px 间隔的灰色条

### 7.4 加载状态

- **骨架屏**：书籍列表用灰色占位块，渐显动画
- **菊花加载**：页面中间，用于搜索、导入等操作
- **下拉刷新**：标准下拉刷新样式，绿色主题色

## 八、交互设计

### 8.1 阅读页手势

| 手势 | 区域 | 行为 |
|------|------|------|
| 单击 | 左侧 1/3 | 上一章 |
| 单击 | 中间 1/3 | 呼出/隐藏菜单 |
| 单击 | 右侧 1/3 | 下一章 |
| 上下滑动 | 全文 | 滚动阅读 |
| 左滑 | 全文 | 下一章 |
| 右滑 | 全文 | 上一章 |
| 长按 | 文字 | 选中文字（复制/分享/添加书签） |

### 8.2 书架页手势

| 手势 | 行为 |
|------|------|
| 下拉 | 刷新（重新扫描本地文件） |
| 长按书籍 | 进入管理模式（勾选删除） |
| 左滑（列表模式时） | 删除 |

### 8.3 页面转场

- 前进：从右侧滑入
- 返回：向右侧滑出
- 底部弹出面板：从底部向上滑入
- 右侧抽屉：从右侧滑入

## 十、适配说明

### 10.1 屏幕适配

- 以 375px 宽度为设计基准（iPhone 尺寸）
- 使用 uni-app 的 `rpx` 单位自动适配不同屏幕
- 三列网格：屏幕宽度 - 左右边距(16px*2) - 列间距(12px*2) = 书籍宽度 * 3
- 大屏手机（如 Plus/Max）适当放大间距和字号

### 10.2 安全区域

- 底部 tab 适配 iPhone 底部安全区域（Home Indicator）
- 阅读页全屏时，注意避开刘海和底部安全区域
- 顶部导航栏适配状态栏高度

### 10.3 Dark Mode

- 阅读页夜间模式独立控制，不跟随系统
- 其他页面（书架、设置等）MVP 阶段不做深色模式，保持浅色主题
