## 系统架构设计文档

## 1. 项目概述

本项目是基于 **uni-app** 框架开发的跨平台小说阅读器 App，支持 iOS 和 Android 双端。采用纯前端架构，所有逻辑在客户端完成，无需后端服务。

### 1.1 核心能力

- 本地 txt 小说导入与阅读
- 在线书源搜索与在线阅读
- 沉浸式阅读体验（多种翻页模式）
- 排版美化与个性化设置
- 书签与阅读进度管理
- 书源配置管理

### 1.2 技术栈

| 分类 | 技术选型 | 版本/说明 |
|------|----------|-----------|
| 框架 | uni-app | Vue 3 + Vite |
| 状态管理 | Pinia | ^2.3.1 |
| 本地数据库 | plus.sqlite | 5+ 原生 API |
| 文件操作 | plus.io | 5+ 原生 API |
| 编码检测 | jschardet | ^3.1.4 |
| 编码转换 | iconv-lite | ^0.7.3 |
| 网络请求 | uni.request | 原生 API |
| HTML 解析 | cheerio | 0.22.0（预留） |
| 国际化 | vue-i18n | ^9.1.9 |
| CSS 预处理器 | Sass | ^1.101.0 |

## 1. 项目概述

本项目是基于 **uni-app** 框架开发的跨平台小说阅读器 App，支持 iOS 和 Android 双端。采用纯前端架构，所有逻辑在客户端完成，无需后端服务。

### 1.1 核心能力

- 本地 txt 小说导入与阅读
- 在线书源搜索与在线阅读
- 沉浸式阅读体验（多种翻页模式）
- 排版美化与个性化设置
- 书签与阅读进度管理
- 书源配置管理

### 1.2 技术栈

| 分类 | 技术选型 | 版本/说明 |
|------|----------|-----------|
| 框架 | uni-app | Vue 3 + Vite |
| 状态管理 | Pinia | ^2.3.1 |
| 本地数据库 | plus.sqlite | 5+ 原生 API |
| 文件操作 | plus.io | 5+ 原生 API |
| 编码检测 | jschardet | ^3.1.4 |
| 编码转换 | iconv-lite | ^0.7.3 |
| 网络请求 | uni.request | 原生 API |
| HTML 解析 | cheerio | 0.22.0（预留） |
| 国际化 | vue-i18n | ^9.1.9 |
| CSS 预处理器 | Sass | ^1.101.0 |
| 类型系统 | TypeScript | ^4.9.4 |

### 1.3 支持平台

- **主要目标**：iOS App / Android App
- **开发调试**：H5
- **可扩展**：微信小程序、支付宝小程序等（需适配）
